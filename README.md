# LabControl - Registro de Devoluções (Front-End)

---
🔗 **Repositório do Back-End (API):** [Acesse aqui](https://github.com/nicolashenrique921/LabControl-RegistroDeDevolucoes)
---

Interface moderna e intuitiva para o sistema de controle de laboratório. Desenvolvida em **React**, a aplicação oferece uma experiência fluida para o registro e consulta de equipamentos devolvidos.

### 🛠 Tecnologias Utilizadas
* **React.js**
* **React Router DOM** (Navegação)
* **Axios** (Consumo de API)
* **SweetAlert2** (Alertas personalizados)
* **CSS3** (Estilização responsiva)

### 📋 Funcionalidades
* **Formulário Inteligente:** Cadastro de devoluções com campos pré-carregados da API.
* **Histórico Dinâmico:** Visualização de todos os registros com opção de edição e exclusão.
* **Filtros em Tempo Real:** Localização rápida de registros por múltiplos parâmetros.
* **UX Melhorada:** Feedback visual após cada ação e limpeza automática de formulário.

### ⚙️ Como rodar
1. Certifique-se de que a API (Back-End) esteja rodando.
2. Instale as dependências: `npm install`
3. Inicie o projeto: `npm start`
